This experiment is a memorization task involving visual images. It will take approximately 25 - 35 minutes to complete.


Next (1 of 2)

You will be presented with 100 objects, one at a time, each presented for 2 seconds.

Study the visual details of these objects when presented.


After every object, you will see the word "remember" or the word "forget".

If "forget" is displayed, this means that you should forget about the previously displayed object.

If "remember" is displayed, you should retain the previously displayed object in memory. You will later be tested on your memory for this object.

The rest of the instructions will be provided when relevant via text pop-ups.
