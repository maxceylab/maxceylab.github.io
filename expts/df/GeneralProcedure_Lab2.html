This experiment is a memorization task involving visual images. It will take approximately 30 - 45 minutes to complete.

You will be presented with 90 objects, one at a time, each presented for 3 seconds.

Study the visual details of these objects for a later memory test.


Memory for objects will need to be as detailed as "white grand piano", for example, simply memorizing "piano" for the above image would not be helpful during testing.

After every object, you will see the word "remember" or the word "forget".

If "forget" is displayed, this means that you should forget about the previously displayed object.

If "remember" is displayed, you should retain the previously displayed object in memory. You will later be tested on your memory for this object.

The rest of the instructions will be provided when relevant via text pop-ups. Please only use your memory for this experiment -- do not use external devices to aid your memory.
